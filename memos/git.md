# Memo Git

- Setup & Init

  > git init
  >
  > git clone [url]

- Stage & Snapshot

  > git add [filename]
  >
  > git add \* (all file)
  >
  > git commit -m [descriptive message]

- Share & Update

  > git push [alias][branch]
  >
  > git pull

- branch & merge

  > git branch
  >
  > git branch [branch-name]
  >
  > git checkout
  >
  > git merge [branch]
  >
  > git log

- tips
  > git config color.ui true
  >
  > git congif format.pretty oneline
